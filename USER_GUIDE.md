# User Guide

## Basic Terms

- Exercise: one movement, such as Bench Press, Squat, or Dumbbell Row.
- Routine: a planned workout made of several exercises.
- Superset: two or more exercises performed back-to-back before resting.
- Set: one round of an exercise.
- Reps: how many times you perform the movement in one set.
- Weight: the load used for the set.
- RIR/RPE: effort ratings used to describe how hard a set felt.
- History: completed workouts saved after finishing.
- Bodyweight as Load: for bodyweight exercises, the app can count your bodyweight as part of the total load.
- Duration Exercise: an exercise tracked by time instead of reps, such as Plank or Hollow Hold.
- Last Performance Prefill: the app can use your previous workout data to prefill reps, weight, or time in a future workout.

## Basic Flow

1. Open the app.
2. Go to Exercises to check or create exercises.
3. Go to Routines to create or select a workout plan.
4. Start a workout from a routine.
5. Log each set as you train.
6. Finish the workout.
7. Check History to review what you completed.

## Creating an Exercise

1. Open Exercises.
2. Tap Add Exercise.
3. Enter the exercise name.
4. Choose the body part and equipment.
5. Choose whether the exercise uses reps or duration.
6. For bodyweight exercises, choose whether bodyweight should count as load.
7. Add exercise notes or setup notes if needed.

## Organizing Exercises

In the Exercises tab, you can organize the exercise list in different ways:

- manually
- by name
- by body part
- by equipment

## Creating a Routine

1. Open Routines.
2. Create a new routine.
3. Add exercises or supersets.
4. Set the number of sets, rep range, rest time, and warm-up sets.
5. Set effort level, tempo, techniques, and routine-specific notes as needed.

## Completing a Workout

1. Open a routine.
2. Tap Start Workout.
3. Enter reps and weight for each set.
4. Tap Log after completing a set.
5. Rest when the timer starts.
6. Record notes about the session if needed.
7. Switch exercises, edit the workout plan, or edit exercise notes if needed.
8. Tap Finish and confirm to save the workout to History.

## Rest Timer

After you log a set, the rest timer starts automatically.

If you close the rest timer overlay while the timer is still running, you can show it again by:

- briefly opening Notification Center / Lock Screen, then returning to the app
- going to the Home Screen, then opening the app again

The timer continues running even if the overlay is dismissed.

## Last Performance Prefill

The app can use previous workout data to help fill in future workouts.

For example, if you previously logged Bench Press for 8 reps at 50 kg, the next workout can show that previous performance so you can log more quickly.

You can control whether a completed workout is used for future prefill:

- when finishing or saving a workout
- later from History

This is useful if a workout was unusual, such as a deload, recovery day, or incomplete session, and you do not want it to affect future workout suggestions.

## Checking History

History shows completed workouts.

Use History to review:

- exercises performed
- sets completed
- weight and reps
- session notes
- progress over time per exercise

## Settings

Use Settings to adjust app defaults and manage data.

You can:

- choose weight unit, such as lb or kg
- choose effort type, such as RIR or RPE
- set your bodyweight, which is used by exercises that count bodyweight as load
- set default sets, rep ranges, and rest times
- import or export exercises
- import routines
- export workout history

# 사용자 가이드

## 기본 용어

- 운동: 벤치프레스, 스쿼트, 덤벨 로우처럼 하나의 운동 동작을 말합니다.
- 루틴: 여러 운동으로 구성된 운동 계획입니다.
- 슈퍼세트: 두 개 이상의 운동을 쉬지 않고 이어서 수행한 뒤 휴식하는 방식입니다.
- 세트: 한 운동을 한 번 수행하는 단위입니다.
- 반복 횟수: 한 세트에서 동작을 몇 번 반복했는지를 말합니다.
- 중량: 해당 세트에서 사용한 무게입니다.
- RIR/RPE: 세트가 얼마나 힘들었는지 기록하는 운동 강도 지표입니다.
- 기록: 운동을 완료한 뒤 저장된 운동 기록입니다.
- 체중 포함 중량: 맨몸 운동에서 자신의 체중을 전체 중량에 포함할 수 있는 설정입니다.
- 시간 기반 운동: 플랭크나 할로우 홀드처럼 반복 횟수 대신 시간으로 기록하는 운동입니다.
- 이전 기록 자동 입력: 이전 운동 기록을 바탕으로 다음 운동에서 반복 횟수, 중량, 시간을 미리 채워 주는 기능입니다.

## 기본 사용 흐름

1. 앱을 엽니다.
2. 운동 탭에서 운동을 확인하거나 새로 추가합니다.
3. 루틴 탭에서 운동 계획을 만들거나 선택합니다.
4. 루틴에서 운동을 시작합니다.
5. 운동하면서 각 세트를 기록합니다.
6. 운동을 종료합니다.
7. 기록 탭에서 완료한 운동을 확인합니다.

## 운동 만들기

1. 운동 탭을 엽니다.
2. 운동 추가를 누릅니다.
3. 운동 이름을 입력합니다.
4. 부위와 장비를 선택합니다.
5. 반복 횟수로 기록할지, 시간으로 기록할지 선택합니다.
6. 맨몸 운동의 경우 체중을 중량에 포함할지 선택합니다.
7. 필요하면 운동 메모나 세팅 메모를 추가합니다.

## 운동 정리하기

운동 탭에서는 운동 목록을 여러 방식으로 정리할 수 있습니다.

- 직접 정렬
- 이름순 정렬
- 부위별 정렬
- 장비별 정렬

## 루틴 만들기

1. 루틴 탭을 엽니다.
2. 새 루틴을 만듭니다.
3. 운동이나 슈퍼세트를 추가합니다.
4. 세트 수, 반복 범위, 휴식 시간, 워밍업 세트를 설정합니다.
5. 필요하면 운동 강도, 템포, 운동 기법, 루틴 전용 메모를 설정합니다.

## 운동 완료하기

1. 루틴을 엽니다.
2. 운동 시작을 누릅니다.
3. 각 세트의 반복 횟수와 중량을 입력합니다.
4. 세트를 완료한 뒤 기록을 누릅니다.
5. 타이머가 시작되면 휴식합니다.
6. 필요하면 세션 메모를 기록합니다.
7. 필요하면 운동을 교체하거나, 운동 계획을 수정하거나, 운동 메모를 수정합니다.
8. 종료를 누른 뒤 확인하면 운동이 기록에 저장됩니다.

## 휴식 타이머

세트를 기록하면 휴식 타이머가 자동으로 시작됩니다.

타이머가 아직 실행 중일 때 휴식 타이머 화면을 닫은 경우, 다음 방법으로 다시 표시할 수 있습니다.

- 알림 센터 / 잠금 화면을 잠깐 열었다가 앱으로 돌아오기
- 홈 화면으로 나갔다가 앱을 다시 열기

타이머 화면을 닫아도 타이머는 계속 진행됩니다.

## 이전 기록 자동 입력

앱은 이전 운동 기록을 바탕으로 다음 운동 입력을 더 쉽게 할 수 있습니다.

예를 들어, 이전에 벤치프레스를 50kg으로 8회 기록했다면, 다음 운동에서 그 기록을 참고할 수 있습니다.

완료한 운동을 이후 자동 입력에 사용할지 선택할 수 있습니다.

- 운동을 종료하거나 저장할 때 설정할 수 있습니다.
- 나중에 기록 탭에서도 변경할 수 있습니다.

이 기능은 디로드, 회복 운동, 미완성 운동처럼 평소 기록으로 사용하고 싶지 않은 운동을 제외할 때 유용합니다.

## 기록 확인하기

기록 탭에서는 완료한 운동을 확인할 수 있습니다.

기록에서 확인할 수 있는 내용:

- 수행한 운동
- 완료한 세트
- 중량과 반복 횟수
- 세션 메모
- 운동별 진행 변화

## 설정

설정에서는 앱 기본값과 데이터를 관리할 수 있습니다.

설정할 수 있는 내용:

- 중량 단위 선택: lb 또는 kg
- 운동 강도 방식 선택: RIR 또는 RPE
- 체중 포함 중량 운동에 사용되는 체중 설정
- 기본 세트 수, 반복 범위, 휴식 시간 설정
- 운동 가져오기 / 내보내기
- 루틴 가져오기
- 운동 기록 내보내기
