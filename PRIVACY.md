# Privacy Policy

Log is a personal iOS gym logging app.

## Data Storage

Log stores workout data locally on the user's device using SwiftData.

This may include:

- exercises
- routines
- workout logs
- set data
- notes entered by the user
- progress history

## Data Collection

Log does not currently collect, transmit, sell, or share personal data.

Log does not currently use:

- user accounts
- cloud sync
- analytics services
- advertising trackers
- third-party login
- external databases

## TestFlight Feedback

If Log is tested through TestFlight, testers may choose to send feedback, screenshots, or crash reports through Apple's TestFlight system.

That feedback is handled through Apple's TestFlight tools and may be visible to the developer for testing and debugging purposes.

## User Control

Workout data is stored on the user's device.

Deleting the app may delete locally stored app data unless the device or system preserves it through backup behavior.

## Contact

For questions about this project, contact the developer through the repository or the TestFlight feedback channel.

## Project Status

Log is currently a beta / portfolio project, not a public App Store release.
